# Task Phase Submission Repository

## Overview

This repository has been created as part of the **Task Phase** for the student project.  
All files and folders present in the **main branch** represent the individual tasks that were assigned during this phase.

Each task file has been:

- Implemented as per the given problem statement  
- Tested for correctness  
- Structured and documented where required  

## Repository Structure

> **Important:**  
> Every file located in the root of the `main` branch corresponds to a **completed task**.

There are no placeholder, demo, or incomplete files in this repository.

/ (main branch)
│
├── Task files → Completed assignments
├── Supporting files → Required dependencies or helpers
└── This README.md

## Usage

You may open and run each file independently based on its task requirements.
